# Korpuslinguistische Analyse des Stadtwächters (1929-1931)

## Rohdaten
Die Texte basieren auf digitalisierten Ausgaben des Stadtwächters.
Die Originale befinden sich im Stadtarchiv Osnabrück.
Die .txt-Dateien sind aus urheberrechtlichen Gründen nicht im 
Repository enthalten.

pip install -r pipeline/requirements.txt
